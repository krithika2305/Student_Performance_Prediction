# 🎓 Student Performance Prediction System

A machine learning-based web application that predicts student academic performance based on various factors including attendance, study habits, mental health, and lifestyle.

---

## 🚀 Features

- 🔍 **Predictive Analysis**  
  Uses a trained Logistic Regression model to predict student performance levels  

- 💻 **User-Friendly Interface**  
  Interactive web form for easy data input  

- ⚡ **Real-time Predictions**  
  Instant results displayed after submission  

- 🧠 **Personalized Insights**  
  Provides tailored suggestions based on predicted performance  

---

## 🛠️ Tech Stack

### 🎨 Frontend
- HTML  
- CSS  
- Bootstrap  

### ⚙️ Backend
- Python (Flask)  

### 🤖 Machine Learning
- Scikit-learn (Logistic Regression)  
- Pandas  
- NumPy  

---

## ⚙️ How It Works

1. User inputs student data through the web form  
2. Data is preprocessed and scaled  
3. Machine Learning model predicts performance level  
4. Result is displayed with recommendations  

---

## 🧠 Machine Learning Details

- **Algorithm Used:** Logistic Regression  

### 📊 Features Used:
- Age  
- Gender  
- Semester  
- Attendance  
- Internal Marks  
- CGPA  
- Study Hours  
- Assignment Completion  
- Mental Health  
- Lifestyle Factors  

- **Model Accuracy:** ~73%
